# AI Powered Text2SQL

Convert natural language into SQL using a Local LLM.

## Tech Stack

- React + TypeScript
- Tailwind CSS
- FastAPI
- PostgreSQL
- SQLAlchemy
- Ollama
- Qwen2.5

## Features

- Natural Language → SQL
- AI Generated SQL Viewer
- PostgreSQL Execution
- Query History
- Export Results to CSV
- Modern Dashboard
- Local AI using Ollama
